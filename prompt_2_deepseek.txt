DESIGN SYSTEM — Follow These Exactly

─────────────────────────────────

​

Color Philosophy

- NEVER use default CSS colors (red, blue, green) or generic hex values (#333, #007bff)

- Use the brand palette from the PRD with strong contrast and visual harmony

- Backgrounds: warm off-whites (#fafaf8, #f5f5f0, #e3e1e2) — NEVER pure white

- Use foreground/70 and foreground/80 opacity for secondary text hierarchy

​

Typography

- Import TWO fonts from Google Fonts:

    Headings: a serif font (Playfair Display, DM Serif Display, Lora, Cormorant Garamond)

    Body: a clean sans-serif (DM Sans, Inter, or Outfit)

- Hero headline: text-7xl on desktop, text-5xl on mobile

- Small uppercase labels: tracking-wide or tracking-[0.3em]

- Headlines: use text-balance to prevent awkward line breaks

- Subtext: always text-muted-foreground, never the same weight as headings

​

Spacing & Shapes

- ALL cards, containers, image wrappers: border-radius 20-24px (rounded-3xl)

- NO sharp corners anywhere on the site

- Section padding: py-24 (96px vertical)

- Content max-width: max-w-7xl with px-6 lg:px-8

- Generous whitespace between every element

ANIMATION SYSTEM

─────────────────────────────────

​

Scroll-Triggered Reveals (IntersectionObserver)

- Cards and images: opacity-0 scale-95 → opacity-100 scale-100

- Duration: 700ms, timing: ease-out

- Stagger each card by 80ms (0ms, 80ms, 160ms...)

- Threshold: 0.1

​

Text Blur-In — apply to all section headings:

​

  @keyframes blur-in {

    0%   { filter: blur(12px); opacity: 0; transform: translateY(8px); }

    100% { filter: blur(0);    opacity: 1; transform: translateY(0);   }

  }

  .animate-blur-in { animation: blur-in 0.8s ease-out forwards; }

​

  Stagger animation-delay: label 0.2s, headline 0.4s, subtext 0.6s

  Elements start opacity-0 and only animate when scrolled into view

​

Hover Micro-Interactions

- Cards: hover:scale-[1.02] with 300ms transition

- Buttons with arrows: group-hover:translate-x-1 on the arrow

- Quick-add buttons: opacity-0 translate-y-2 → group-hover:opacity-100 translate-y-0

- All interactive elements: transition-all duration-300 ease-out
